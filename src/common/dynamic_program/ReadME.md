# 动态规划

DSA（数据结构分析）设计：

Make it work

Make it right

Make it fast

递归章节已经谈到过，当递归实例存在很多重复的计算时，往往计算复杂度会很高，导致算法不是理想的算法。而动态规划则是将递归方向颠倒，从而避免了递归重复实例的产生，所以动态规划天然的能够避免递归问题中的计算爆炸。

## 使用动态规划解决Fib问题

**算法**：每一次都使用两个变量保存前两步计算的结果，不断更新，最终得到需要求的Fib(n)

```cpp
def DynFib(int n)
{
    int f1 = 0;
    int f2 = 1;
    while (0 < n--) {
        f2 = f1 + f2
        f1 = f2 - f1
    }
    return f2;
}
```
